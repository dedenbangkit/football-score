# Football LiveScore
Livescore Scrape

## Requirements 
Download [Chrome WebDriver](http://chromedriver.chromium.org/downloads) and save it in the app directory.

## Installation
```
$ git clone https://github.com/dedenbangkit/football-score.git
$ cd football-score
$ pip install -r requirements.txt
```

## Dependencies
- Flask 0.12
- Selenium 3.11.0
